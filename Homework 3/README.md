# Welcome to your To-Do Expo app

This is an [Expo](https://expo.dev) project created with [`npx create-expo-app {app-name} --template blank`](https://www.npmjs.com/package/create-expo-app).


# Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```
